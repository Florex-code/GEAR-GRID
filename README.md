# GEAR-GRID

Premium Car Marketplace built with React + Vite.

## Features
- 24+ luxury cars with Unsplash images
- Admin dashboard for inventory management
- Favorites/wishlist system
- Dark/light theme toggle
- Responsive design

## Tech Stack
- React 19
- Vite
- Framer Motion
- React Router
- Unsplash API
